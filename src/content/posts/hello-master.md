在這個資訊爆炸的時代，我選擇建立一個屬於自己的小角落——一個遠離演算法推薦、不受平台限制的獨立空間。

這裡是我記錄技術探索、學習心得與生活隨想的地方。

| 主題 | 說明 |

### 技術棧

本站採用現代化的靜態網站架構：

- **框架**：Astro 5.x + Svelte + Tailwind CSS
- **部署**：Cloudflare Pages (全球邊緣 CDN)
- **資料**：Cloudflare D1 + Workers
- **主題**：基於 Fuwari 深度客製化

---

## 建站歷程

從零開始建立這個部落格，經歷了幾個重要的里程碑：

1. **專案遷移**：將開發環境整合至本地，方便版本控制
2. **環境部署**：選用 pnpm 作為套件管理器，確保依賴一致性
3. **視覺設計**：設計專屬 Logo、橫幅圖片，建立品牌識別
4. **功能開發**：整合邊緣計數、分類標籤、搜尋等功能
5. **內容規劃**：制定文章分類與標籤體系

## 為什麼選擇獨立部落格？

在 Medium、掘金、知乎等平台盛行的今天，為什麼還要自己架設部落格？

| 考量 | 平台 | 獨立部落格 |
|------|------|-----------|
| **內容所有權** | 平台持有 | 完全自有 |
| **排版自由度** | 受限 | 完全控制 |
| **數據隱私** | 被追蹤 | 自行管理 |
| **遷移成本** | 高 | 低 |
| **SEO 控制** | 有限 | 完全掌握 |
| **商業限制** | 受平台規則約束 | 無 |

對我來說，獨立部落格不只是一個發布內容的地方，更是一個技術實踐的場域。從 DNS 設定到 CI/CD 部署，從前端開發到邊緣運算，每一個環節都是學習的機會。
